# Google Sheets (đã ngừng dùng)

Dự án **không còn** đọc/ghi phim qua Google Sheets. Nguồn chuẩn:

- **Admin + API:** Supabase (`movies`, `movie_episodes`) — xem [../supabase/schema-movies-episodes.sql](../supabase/schema-movies-episodes.sql).
- **Build:** Supabase (khi có env) hoặc `custom_movies.xlsx` — xem [../custom-movies-template.md](../custom-movies-template.md).

Thư mục này chỉ giữ **CSV mẫu** (`movies-template.csv`, `episodes-template.csv`) để tham khảo cột tương thích với bảng Supabase.
