# CSV mẫu cột phim / tập

Dự án lưu phim tùy chỉnh qua **Supabase** (`movies`, `movie_episodes`) hoặc file Excel `custom_movies.xlsx` khi build — xem [../custom-movies-template.md](../custom-movies-template.md).

Thư mục này giữ **CSV mẫu** (`movies-template.csv`, `episodes-template.csv`) để tham khảo cột tương thích với bảng Supabase (import hoặc nhập tay).
